P/Z Method: for volumetric dry-gas reservoirs, P/Z plotted against Gp is linear; extrapolating to P/Z=0 gives IGIP. Water influx (We) causes upward curvature/deviation from the straight line — Havlena-Odeh material balance solves for We using an aquifer influx function so the corrected plot straightens again.
